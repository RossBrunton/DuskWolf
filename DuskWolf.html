On all layers -=- ARROW: Move frame | CTRL+ARROW: Change layer | ++ARROW: Extend map | -+ARROW: Shrink map | S: Save (to console) | G: Go to another room
On tilemap layers -=- A+ARROW: Adjust alpha | F+ARROW: Resize frame | SHIFT+ARROW: Change tile | I: Set source image | E: Edit animation
On entity layer -=- A: Pick up/put down | SHIFT+ARROW: Move slightly | DEL: Delete entity | C: Clear all entities | 0-9: Drop defined entity | SHIFT+0-9: Set defined entry | E: Show/Hide entity list | N: Name next entity | W: Open workshop
On transition layer -=- A: Add a new transition | R: Remove a transition | I: Edit transition in
